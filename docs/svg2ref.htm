class SVGg

The methods listed below can be called on the SVG2 instance or any group (SVG2g) instance to create basic shapes that correspond to a single element in the SVG drawing. These methods return a jquery collection representing the element.

  • circle
  • ellipse
  • line
  • path
  • poly
  • rect
  • star
  • text

These methods add images to your drawing. Because the browser loads images asynchronously, they return a Promise rather than a jquery collection.

  • image
  • mjax

These methods produce composite shapes. They typically return a group (an instance of SVG2g) usually composed of more than one child element.

  • arrow
  • chevron
  • coil
  • cylinder
  • edot
  • energy_flow
  • error_bar_x
  • error_bar_y
  • graph
  • label
  • locus
  • plot
  • pm
  • ray
  • ruler
  • stickman
  • tip_to_tail
