class SVG2

Drawing Methods

The methods listed below can be called on the SVG2 instance or any group (SVG2group) instance to create basic shapes that correspond to a single element in the SVG drawing. These methods return a jquery collection representing the element.

  • circle
  • ellipse
  • line
  • path
  • poly
  • rect
  • star

These methods add images to your drawing. Because the browser loads images asynchronously, they return a Promise rather than a jquery collection.

  • image
  • mjax

These methods are used to create more complicated shapes and typically return a group (an instance of SVG2group) often containing more than one child element.

  • arrow
  • chevron
  • coil
  • cylinder
  • edot
  • energy_flow
  • error_bar_x
  • error_bar_y
  • locus
  • plot
  • pm
  • ray
  • ruler
  • stickman
  • text
  • ticks
  • tip_to_tail

There are also static functions that create an entire drawing and return a new SVG2 instance...

  • ebg
  • vec_diag
